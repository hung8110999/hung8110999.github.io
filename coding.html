3 Data Structures

‚ñº

Data structures are the building blocks of efficient programs.

  • Arrays & Linked Lists: Sequential and node-based storage
  • Stacks & Queues: LIFO and FIFO structures
  • Hash Tables: Key-value pairs for fast lookups
  • Trees: Binary trees, BST, AVL trees
  • Graphs: Adjacency list/matrix, traversals (BFS, DFS)

4 Algorithms

‚ñº

Algorithms are step-by-step procedures for solving problems efficiently.

  • Sorting: Bubble, Merge, Quick, Heap sort
  • Searching: Binary search, linear search
  • Dynamic Programming: Memoization, tabulation
  • Greedy Algorithms: Optimal local choices
  • Big O Notation: Time & space complexity analysis

Key Insight: Always analyze time complexity before choosing an algorithm. O(n log n) sorting is much better than O(n¬≤) for large datasets!
